# &lt;app-icon&gt;

A web component used as a wrapper for an icon, providing click-listener and custom `icon-clicked`-event.

## Events

| Event Name | Fired When |
|------------|------------|
| `icon-clicked`| The icon is clicked. |

## Example

```html
   <app-icon>
     <img src="/my-icon.png">
   </app-icon>
```
